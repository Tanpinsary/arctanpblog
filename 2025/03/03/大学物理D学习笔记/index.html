质点运动学

一、基本物理量 (微分定义)

  • 位矢:

    • 矢量形式: r⃗=r⃗(t)\vec{r} = \vec{r}(t)r=r(t)
    • 直角坐标形式: r⃗=xi^+yj^+zk^\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}r=xi^+yj^​+zk^ 或 r⃗=x(t)i^+y(t)j^+z(t)k^\vec{r} = x(t)\hat{i} + y(t)\hat{j} + z(t)\hat{k}r=x(t)i^+y(t)j^​+z(t)k^

  • 位移:

    • 定义: Δr⃗=r⃗B−r⃗A\Delta\vec{r} = \vec{r}_B - \vec{r}_AΔr=rB​−rA​
    • 直角坐标形式: Δr⃗=Δxi^+Δyj^\Delta\vec{r} = \Delta x\hat{i} + \Delta y\hat{j}Δr=Δxi^+Δyj^​

  • 速度:

    • 平均速度: v⃗ˉ=Δr⃗Δt\bar{\vec{v}} = \frac{\Delta\vec{r}}{\Delta t}vˉ=ΔtΔr​
    • 瞬时速度 (速度): v⃗=lim⁡Δt→0Δr⃗Δt=dr⃗dt\vec{v} = \lim_{\Delta t\to0}\frac{\Delta\vec{r}}{\Delta t} = \frac{d\vec{r}}{dt}v=limΔt→0​ΔtΔr​=dtdr​
    • 直角坐标形式: v⃗=dxdti^+dydtj^+dzdtk^\vec{v} = \frac{dx}{dt}\hat{i} + \frac{dy}{dt}\hat{j} + \frac{dz}{dt}\hat{k}v=dtdx​i^+dtdy​j^​+dtdz​k^
      • 分量: vx=dxdtv_x = \frac{dx}{dt}vx​=dtdx​, vy=dydtv_y = \frac{dy}{dt}vy​=dtdy​, vz=dzdtv_z = \frac{dz}{dt}vz​=dtdz​

  • 速率:

    • 瞬时速率 (速率): v=lim⁡Δt→0ΔsΔt=dsdtv = \lim_{\Delta t\to0}\frac{\Delta s}{\Delta t} = \frac{ds}{dt}v=limΔt→0​ΔtΔs​=dtds​
    • 速度大小等于速率: ∣v⃗∣=∣dr⃗dt∣=dsdt=v|\vec{v}| = |\frac{d\vec{r}}{dt}| = \frac{ds}{dt} = v∣v∣=∣dtdr​∣=dtds​=v

    • 直角坐标形式: a⃗=dvxdti^+dvydtj^+dvzdtk^\vec{a} = \frac{dv_x}{dt}\hat{i} + \frac{dv_y}{dt}\hat{j} + \frac{dv_z}{dt}\hat{k}a=dtdvx​​i^+dtdvy​​j^​+dtdvz​​k^
      • 分量: ax=dvxdta_x = \frac{dv_x}{dt}ax​=dtdvx​​, ay=dvydta_y = \frac{dv_y}{dt}ay​=dtdvy​​, az=dvzdta_z = \frac{dv_z}{dt}az​=dtdvz​​

二、运动学的两类问题 (积分形式)

  • 由加速度求速度 (第二类问题):

    • v⃗(t)=v⃗(0)+g⃗t\vec{v}(t) = \vec{v}(0) + \vec{g}tv(t)=v(0)+g​t (针对重力场)

四、相对运动 (伽利略变换)

  • 核心：绝对 = 相对 + 牵连

    • 物体相对于S系的运动——绝对运动；
    • 物体相对于S’系的运动——相对运动；
    • S’系相对于S系的运动——牵连运动。

  • 位矢变换: r⃗绝对=r⃗相对+r⃗牵连\vec{r}_{\text{绝对}} = \vec{r}_{\text{相对}} + \vec{r}_{\text{牵连}}r绝对​=r相对​+r牵连​

  • 速度变换: v⃗绝对=v⃗相对+v⃗牵连\vec{v}_{\text{绝对}} = \vec{v}_{\text{相对}} + \vec{v}_{\text{牵连}}v绝对​=v相对​+v牵连​ (或 v⃗=v⃗′+v⃗0\vec{v} = \vec{v}' + \vec{v}_0v=v′+v0​)

  • 加速度变换: a⃗绝对=a⃗相对+a⃗牵连\vec{a}_{\text{绝对}} = \vec{a}_{\text{相对}} + \vec{a}_{\text{牵连}}a绝对​=a相对​+a牵连​ (或 a⃗=a⃗′+a⃗0\vec{a} = \vec{a}' + \vec{a}_0a=a′+a0​)

    • 若S’系相对S系做匀速直线运动: a⃗=a⃗′\vec{a} = \vec{a}'a=a′

质点动力学

一、牛顿运动定律

1. 牛顿第一定律

  • 质点保持静止或匀速直线运动的状态，除非受到外力作用。
    • 合外力为零：∑F‾i=0\sum\overline{F}_{i}=0∑Fi​=0
    • 速度恒定：v⃗=const\vec{v}=constv=const

2. 牛顿第二定律

  • 力的作用改变物体的运动状态。
    • 当质量为常量时：F‾=ma‾\overline{F}=m\overline{a}F=ma
    • 普遍形式：F‾=d(mυ‾)dt\overline{F}=\frac{d(m\overline{\upsilon})}{dt}F=dtd(mυ)​
    • 完整形式：F‾=ma⃗=mdυ‾dt=md2r‾dt2\overline{F}=m\vec{a}=m\frac{d\overline{\upsilon}}{dt}=m\frac{d^{2}\overline{r}}{dt^{2}}F=ma=mdtdυ​=mdt2d2r​
  • 直角坐标系分量式:
    • Fx=max=mdυxdt=md2xdt2F_{x}=ma_{x}=m\frac{d\upsilon_{x}}{dt}=m\frac{d^{2}x}{dt^{2}}Fx​=max​=mdtdυx​​=mdt2d2x​
    • Fy=may=mdυydt=md2ydt2F_{y}=ma_{y}=m\frac{d\upsilon_{y}}{dt}=m\frac{d^{2}y}{dt^{2}}Fy​=may​=mdtdυy​​=mdt2d2y​
    • Fz=maz=mdυzdt=md2zdt2F_{z}=ma_{z}=m\frac{d\upsilon_{z}}{dt}=m\frac{d^{2}z}{dt^{2}}Fz​=maz​=mdtdυz​​=mdt2d2z​
  • 自然坐标系分量式:
    • 切向力：Ft=mat=mdυdtF_{t}=ma_{t}=m\frac{d\upsilon}{dt}Ft​=mat​=mdtdυ​
    • 法向力：Fn=man=mv2ρF_{n}=ma_{n}=m\frac{v^{2}}{\rho}Fn​=man​=mρv2​

3. 牛顿第三定律

  • 两个物体间的相互作用力总是大小相等、方向相反，作用在同一条直线上。
    • F⃗12=−F⃗21\vec{F}_{12}=-\vec{F}_{21}F12​=−F21​

二、常见力与摩擦力

1. 万有引力

  • 任意两个质点之间的相互吸引力：F=Gm1m2r2F=G\frac{m_{1}m_{2}}{r^{2}}F=Gr2m1​m2​​
    • 万有引力常量：G=6.67×10−11N⋅m2⋅kg−2G=6.67\times10^{-11}N\cdot m^{2}\cdot kg^{-2}G=6.67×10−11N⋅m2⋅kg−2
  • 重力：P=mgP=mgP=mg

  • 胡克定律（弹簧）：f=−kxf=-kxf=−kx

3. 摩擦力

  • 静摩擦力：F‾s=−F‾\overline{F}_{s}=-\overline{F}Fs​=−F

三、功与能

1. 功

  • 变力的功：W=∫abF‾⋅dr⃗W=\int_{a}^{b}\overline{F}\cdot d\vec{r}W=∫ab​F⋅dr

    • W=Ek2−Ek1=12mv22−12mv12W=E_{k2}-E_{k1}=\frac{1}{2}mv_{2}^{2}-\frac{1}{2}mv_{1}^{2}W=Ek2​−Ek1​=21​mv22​−21​mv12​
  • 质点系动能定理: 作用于质点系的外力和内力所做功的总和，等于系统总动能的增量。
    • W外+W内=Ek−Ek0W_{外}+W_{内}=E_{k}-E_{k0}W外​+W内​=Ek​−Ek0​
    • 一对内力做功之和：dW=F⃗2⋅dr⃗′dW=\vec{F}_{2}\cdot d\vec{r}^{\prime}dW=F2​⋅dr′ (其中 dr⃗′d\vec{r}^{\prime}dr′ 为相对位移)

4. 势能

  • 保守力做功等于势能减少量：W保守力=−(Ep2−Ep1)=−ΔEpW_{保守力}=-(E_{p2}-E_{p1})=-\Delta E_{p}W保守力​=−(Ep2​−Ep1​)=−ΔEp​

    • 分量形式：Fx=−∂Ep∂x,Fy=−∂Ep∂y,Fz=−∂Ep∂zF_{x}=-\frac{\partial E_{p}}{\partial x}, F_{y}=-\frac{\partial E_{p}}{\partial y}, F_{z}=-\frac{\partial E_{p}}{\partial z}Fx​=−∂x∂Ep​​,Fy​=−∂y∂Ep​​,Fz​=−∂z∂Ep​​

四、功能原理与机械能守恒

  • 功能原理: 外力和非保守内力对系统做功之和，等于系统机械能的增量。
    • W外+W非保守内力=E−E0=ΔEW_{外}+W_{非保守内力}=E-E_{0}=\Delta EW外​+W非保守内力​=E−E0​=ΔE
    • 机械能：E=Ek+EpE=E_{k}+E_{p}E=Ek​+Ep​
  • 机械能守恒定律: 如果一个系统只有保守内力做功，其他外力和非保守内力不做功，则系统的机械能保持不变。
    • E=Ek+Ep=常量E = E_k + E_p = \text{常量}E=Ek​+Ep​=常量
    • ΔEk=−ΔEp\Delta E_{k}=-\Delta E_{p}ΔEk​=−ΔEp​

五、动量

  • 动量：p⃗=mν⃗\vec{p}=m\vec{\nu}p​=mν
  • 冲量：I‾=∫t1t2F⃗dt\overline{I}=\int_{t_{1}}^{t_{2}}\vec{F}dtI=∫t1​t2​​Fdt

  • 质点动量定理: 作用在质点上的合力的冲量，等于质点动量的增量。
    • I⃗=p⃗2−p⃗1\vec{I}=\vec{p}_{2}-\vec{p}_{1}I=p​2​−p​1​
    • 微分形式：F⃗dt=dp⃗\vec{F}dt=d\vec{p}Fdt=dp​
  • 质点系动量定理: 作用于质点系的合外力的冲量，等于系统总动量的增量。
    • ∫t1t2F‾外dt=p⃗−p‾0\int_{t_{1}}^{t_{2}}\overline{F}_{外}dt=\vec{p}-\overline{p}_{0}∫t1​t2​​F外​dt=p​−p​0​
    • 微分形式：ΣF⃗外=dp⃗总dt\Sigma\vec{F}_{外}=\frac{d\vec{p}_{总}}{dt}ΣF外​=dtdp​总​​

3. 动量守恒定律

  • 当系统所受合外力为零时，系统的总动量保持不变。
    • 条件：ΣF外=0\Sigma F_{外}=0ΣF外​=0
    • 结论：P⃗总=∑miv⃗i=\vec{P}_{总}=\sum m_{i}\vec{v}_{i}=P总​=∑mi​vi​= 常矢量
  • 分量守恒: 若某一方向的合外力为零，则该方向的总动量分量守恒。
    • 例如x方向：ΣFix=0⇒Px=∑mivix=\Sigma F_{ix}=0\Rightarrow P_{x}=\sum m_{i}v_{ix}=ΣFix​=0⇒Px​=∑mi​vix​= 常量

六、质心

1. 质心位置

  • 离散质点系：r⃗c=∑mir⃗i∑mi\vec{r}_{c}=\frac{\sum m_{i}\vec{r}_{i}}{\sum m_{i}}rc​=∑mi​∑mi​ri​​

    • 分量形式：xc=1m∫xdmx_{c}=\frac{1}{m}\int xdmxc​=m1​∫xdm, yc=1m∫ydmy_{c}=\frac{1}{m}\int ydmyc​=m1​∫ydm, zc=1m∫zdmz_{c}=\frac{1}{m}\int zdmzc​=m1​∫zdm

2. 质心运动定理

  • 质点系的质心运动，如同一个质量等于系统总质量、受力等于系统所受全部外力的合力的质点的运动。质心的运动与内力无关。
    • F⃗外=m总a⃗C\vec{F}_{外}=m_{总}\vec{a}_{C}F外​=m总​aC​
  • 质心运动与动量守恒:
    • 若合外力为零（动量守恒），则质心加速度为零，质心保持静止或做匀速直线运动。

**静电学 **
