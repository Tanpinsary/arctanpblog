一、行列式

  1. 行列式的计算

    • 行列式展开定理
    • 逆序数

  2. 行列式的性质

    • 不变性
      • 转置 将第 iii 行转变为第 iii 列
      • 倍加 某行（列）加上另一行（列）的 kkk 倍
    • 改变性质
      • 倍乘 从一行（列）中提取公因子
      • 交换 交换两行（列），注意变号
      • 拆分 对于某一行（列）拆成两个行列式的和
    • 归零性质
      • 有零行（列）
      • 有成比例的行（列）
  3. ∣A∣≠0|A| \neq 0∣A∣=0 意味着什么
    • 矩阵
      • r(A)=nr(A) = nr(A)=n
      • 矩阵可逆
    • 方程组
      • 有唯一解
    • 向量组
      • 线性无关
    • 特征值与特征向量
      • 不含 0 特征值

  4. 特殊类型的行列式（计算）

    • 行（列）和相等

      加到同一行，提取因子，用该行（列）将其化为主对角型行列式

    • 爪形行列式

      用中间的爪子依次倍加，知道化为主对角型行列式

三、向量组

  1. 具体向量组的相关\无关
    • 相关：r(a1,a2,...,am)<mr(a_1, a_2, ..., a_m) < mr(a1​,a2​,...,am​)<m
    • 无关：r(a1,a2,...,am)=mr(a_1, a_2, ..., a_m) = mr(a1​,a2​,...,am​)=m
  2. 抽象向量组的相关\无关
    • 相关：若存在不全为零的数 k1,k2,...,kmk_1, k_2, ..., k_mk1​,k2​,...,km​，使得 k1a1+k2a2+...+kmam=0k_1 a_1 + k_2 a_2 + ... + k_m a_m = 0k1​a1​+k2​a2​+...+km​am​=0
    • 无关：当且仅当 k1=k2=...=kmk_1 = k_2 = ... = k_mk1​=k2​=...=km​，才有 k1a1+k2a2+...+kmam=0k_1 a_1 + k_2 a_2 + ... + k_m a_m = 0k1​a1​+k2​a2​+...+km​am​=0
  3. 极大线性无关组 与 线性表出
    • 把向量组写成列向量组，通过多次初等行变换，变化为行最简型矩阵，主元为 1 的向量构成极大无关组，不属于极大无关组的向量由极大无关组线性表出

  • 无穷多解 r(A)=r(A,b)<nr(A) = r(A,b) < nr(A)=r(A,b)<n
