线性代数复习提纲

一、行列式

1. 行列式的计算

  • 行列式展开定理
  • 逆序数

2. 行列式的性质

不变性

  • 转置：将第 iii 行转变为第 iii 列
  • 倍加：某行（列）加上另一行（列）的 kkk 倍

改变性质

  • 倍乘：从一行（列）中提取公因子
  • 交换：交换两行（列），注意变号
  • 拆分：对于某一行（列）拆成两个行列式的和

归零性质

  • 有零行（列）
  • 有成比例的行（列）

3. ∣A∣≠0|A| \neq 0∣A∣=0 意味着什么

  • 矩阵：r(A)=nr(A) = nr(A)=n，矩阵可逆
  • 方程组：有唯一解
  • 向量组：线性无关
  • 特征值与特征向量：不含 000 特征值

4. 特殊类型的行列式（计算）

行（列）和相等

加到同一行，提取因子，用该行（列）将其化为主对角型行列式

爪形行列式

用中间的爪子依次倍加，直到化为主对角型行列式

二、矩阵

1. 矩阵的运算

  • 对加法和数乘满足线性性质
  • 矩阵的乘法：注意内标相同，对左侧行向量和右侧列向量进行求内积的操作

三、向量组

1. 具体向量组的线性相关性

  • 相关：r(a1,a2,…,am)<mr(a_1, a_2, \ldots, a_m) < mr(a1​,a2​,…,am​)<m
  • 无关：r(a1,a2,…,am)=mr(a_1, a_2, \ldots, a_m) = mr(a1​,a2​,…,am​)=m

2. 抽象向量组的线性相关性

  • 相关：存在不全为零的 k1,k2,…,kmk_1, k_2, \ldots, k_mk1​,k2​,…,km​ 使 k1a1+k2a2+⋯+kmam=0k_1 a_1 + k_2 a_2 + \cdots + k_m a_m = 0k1​a1​+k2​a2​+⋯+km​am​=0
  • 无关：仅当 k1=k2=⋯=km=0k_1 = k_2 = \cdots = k_m = 0k1​=k2​=⋯=km​=0 时成立

将向量组写成列向量进行初等行变换，行最简型中主元列构成极大无关组

四、方程组

非齐次方程组解的情况

  • 唯一解：r(A)=r(A,b)=nr(A) = r(A,b) = nr(A)=r(A,b)=n
  • 无穷多解：r(A)=r(A,b)<nr(A) = r(A,b) < nr(A)=r(A,b)<n
  • 无解：r(A)<r(A,b)r(A) < r(A,b)r(A)<r(A,b)

齐次方程组解的情况

  • 零解：r(A)=nr(A) = nr(A)=n
  • 无穷多解：r(A)<nr(A) < nr(A)<n，基础解系个数为 n−r(A)n - r(A)n−r(A)

五、相似

1. 特征值与特征向量

特征方程

det⁡(A−λE)=0\det(A - \lambda E) = 0det(A−λE)=0

特征值的性质

  • 特征值对应矩阵变换 AAA，特征向量不变方向
  • 矩阵 kAkAkA 的特征值为 kλk\lambdakλ，特征向量不变
  • 矩阵 AmA^mAm 的特征值为 λm\lambda^mλm，特征向量不变
  • 矩阵 A−1A^{-1}A−1 的特征值为 1λ\frac{1}{\lambda}λ1​，特征向量不变

特征值的数量性质

  • 所有特征值之和等于矩阵的迹：∑λi=tr(A)\sum \lambda_i = \text{tr}(A)∑λi​=tr(A)
  • 所有特征值的乘积等于行列式：∏λi=∣A∣\prod \lambda_i = |A|∏λi​=∣A∣

2. 相似对角化

条件：充要条件是存在 nnn 个线性无关的特征向量，矩阵可对角化

可对角化的充分条件

  • 实对称矩阵必可对角化
  • 秩为 nnn 的矩阵
  • 特征值互异的矩阵

正交对角化

对于实对称矩阵，存在正交矩阵 QQQ 使得 QTAQ=ΛQ^T A Q = \LambdaQTAQ=Λ，其中 Λ\LambdaΛ 是对角矩阵

六、二次型

1. 二次型的定义与矩阵表示

二次型是关于多个变量的二次齐次多项式，可以表示为矩阵形式：

f(x1,x2,…,xn)=xTAxf(x_1, x_2, \ldots, x_n) = x^T A xf(x1​,x2​,…,xn​)=xTAx

其中 x=(x1,x2,…,xn)Tx = (x_1, x_2, \ldots, x_n)^Tx=(x1​,x2​,…,xn​)T，AAA 是对称矩阵

2. 二次型的标准化方法

配方法：逐步配方，最后化为标准型 f=d1y12+d2y22+⋯+dnyn2f = d_1 y_1^2 + d_2 y_2^2 + \cdots + d_n y_n^2f=d1​y12​+d2​y22​+⋯+dn​yn2​

正交变换法：通过正交变换 x=Pyx = Pyx=Py 将二次型化为标准型

f=λ1y12+λ2y22+⋯+λnyn2f = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2f=λ1​y12​+λ2​y22​+⋯+λn​yn2​

其中 λi\lambda_iλi​ 是对称矩阵 AAA 的特征值，PPP 的列向量是对应的正交特征向量

3. 惯性定理

二次型通过不同的可逆变换化为的标准型中，正项的个数（正惯性指数）和负项的个数（负惯性指数）保持不变，与所用变换无关

记号：设正惯性指数为 ppp，负惯性指数为 qqq，则 p+q≤np + q \leq np+q≤n

4. 规范型

任何二次型都可以化为规范型：

f=z12+⋯+zp2−zp+12−⋯−zp+q2f = z_1^2 + \cdots + z_p^2 - z_{p+1}^2 - \cdots - z_{p+q}^2f=z12​+⋯+zp2​−zp+12​−⋯−zp+q2​

其中 ppp 是正惯性指数，qqq 是负惯性指数

5. 正定二次型的判定

定义：若对所有 x≠0x \neq 0x=0，都有 f(x)>0f(x) > 0f(x)>0，则二次型为正定

判定条件（等价）

  • 所有顺序主子式均为正
  • 矩阵 AAA 的所有特征值都为正
  • 矩阵 AAA 可表示为 A=BTBA = B^T BA=BTB，其中 BBB 为满秩矩阵（Cholesky 分解）
  • 规范型中不含负项

常见二次曲面的分类

  • 椭球面：x2a2+y2b2+z2c2=1\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1a2x2​+b2y2​+c2z2​=1
