自动判卷系统

C++17 开发的极简在线考试后端系统，集成了 AI OCR 评分和在线代码评判功能。

技术栈

  • Web 框架: Crow v1.0+ (header-only)
  • 数据库: SQLite3 with SQLiteCpp wrapper
  • HTTP 客户端: cpr (C++ Requests Library)
  • JSON 处理: nlohmann/json
  • AI OCR: DeepSeek API
  • 代码评判: Judge0 API
  • 构建系统: CMake

核心功能

✅ 用户管理

  • 用户注册和认证
  • 用户信息管理

✅ 试卷管理

  • 试卷列表获取
  • 支持多种题型（选择题、填空题、编程题）

✅ 答题卡 OCR 评分

  • 基于 DeepSeek API 的智能图像识别
  • 自动答案提取和评分
  • 支持单选题、多选题、填空题
  • 详细的评分标准和标准答案

✅ 编程题自动评分

  • 基于 Judge0 API 的代码执行和评分
  • 支持多种编程语言（C++、Python、Java、JavaScript）
  • 包含编译错误检测、运行时错误检测
  • 代码相似性检查（防抄袭）
  • 实时代码执行结果反馈

✅ 成绩管理

  • 实时分数计算
  • 管理员排名查看
  • 提交记录管理

✅ API 接口

  • RESTful API 设计
  • JSON 数据格式
  • CORS 支持

项目结构

API 端点

基础端点

  • GET /health - 健康检查
  • GET /api - API 文档

用户相关

  • POST /api/register - 用户注册

试卷相关

  • GET /api/papers - 获取试卷列表

答题提交

  • POST /api/submit/ocr/{paper_id}/{user_id} - 提交答题卡（OCR评分）

    • 请求体：图片数据（支持 base64 或 multipart/form-data）

  • POST /api/submit/code/{paper_id}/{user_id} - 提交编程代码

管理员功能

  • GET /api/admin/ranking/{paper_id} - 查看试卷排名

评分系统

OCR 评分（答题卡）

  • 单选题: 6题，每题5分
  • 多选题: 3题，每题5分（多选、错选、漏选均不得分）
  • 填空题: 5题，每题5分
  • 总分: 70分

编程题评分

基于 Judge0 API 的执行结果：

  • 通过 (Accepted): 30分
  • 5个测试点，每个测试点6分
